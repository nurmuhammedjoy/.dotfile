## üé® Layouts and Splits

<div align="center">

Master your workspace organization:

</div>

### Window Splitting
- `Alt+h` ‚Üí **Split window horizontally**
- `Alt+v` ‚Üí **Split window vertically**

### Layout Management
- `Alt+f` ‚Üí **Toggle fullscreen mode**
- `Alt+w` ‚Üí **Switch to tabbed layout**
- `Alt+s` ‚Üí **Toggle between split and stacking layouts**

### Advanced Controls
- `Alt+Shift+Space` ‚Üí **Toggle floating mode** for the focused window
- `Alt+Shift+t` ‚Üí **Toggle focus mode** (tree/tiling)
- `Alt+a` ‚Üí **Focus parent container** in layout hierarchy
